Paste tool

Paste something here to save it.

Paste to API from commandline

Paste directly to API

Using curl & intermediary script:

curl -s https://dev.renner.id/dot/paste | sh -s $FILE

Using curl only:

curl -H "Content-Type: text/plain" --data-binary $FILE https://dev.renner.id/p/

Download script

Download using curl:

curl -s https://dev.renner.id/dot/paste > ./paste

Download using wget:

wget https://dev.renner.id/dot/paste

Make runnable:

chmod +x paste

Paste to API

From file:

./paste $FILE

From stdin(piped):

cat $FILE | ./paste

Use globally on machine

Save as user script(paste is already used, so rename):

sudo mv paste /usr/local/bin/paster

Script can now be run anywhere like this:

paster $FILE
